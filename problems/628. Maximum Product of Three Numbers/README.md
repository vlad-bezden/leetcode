** [628. Maximum Product of Three Numbers](https://leetcode.com/problems/maximum-product-of-three-numbers/)
*** Easy

Given an integer array nums, find three numbers whose product is maximum and return the maximum product.

__Example 1:__
```
Input: nums = [1,2,3]
Output: 6
```
__Example 2:__
```
Input: nums = [1,2,3,4]
Output: 24
```

__Example 3:__
```
Input: nums = [-1,-2,-3]
Output: -6
```

__Constraints:__
*3 <= nums.length <= 104
*-1000 <= nums[i] <= 1000
